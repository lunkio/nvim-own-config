# Neovim Own Config

My personal Neovim configuration, tailored for efficient coding, with a focus on plugins for productivity, syntax highlighting, and UI enhancements.

## Configuration Structure

- `init.lua`: Entry point, loads core settings and plugins.
- `lua/`: Contains modular configurations:
- `plugins/`: Plugin specifications (e.g., `blink.cmp.lua`).
- `keymaps.lua`: Custom key bindings.
- `options.lua`: General Neovim options (e.g., tab settings, colors).
- `lsp/`: Lsp configuration(e.g. `lua_ls.lua`).
